# チケット#05: 敵キャラクター拡張

## 📋 概要
中型機と大型機の敵キャラクターを実装する。
敵からの攻撃（弾）システムを構築し、より複雑なゲームプレイを実現する。

## 🎯 完了条件
- 中型機が実装され、自機狙い弾を撃つ
- 大型機が実装され、弾幕攻撃を行う
- 敵弾の管理システムが動作する
- 敵弾が画面外で自動削除される
- 各敵タイプが適切なHPと報酬を持つ

## 📦 依存関係
- チケット#04（基本敵キャラクター）が完了していること
